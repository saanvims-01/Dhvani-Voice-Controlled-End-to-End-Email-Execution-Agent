# Dhvani-Voice-Controlled-End-to-End-Email-Execution-Agent
Dhvani is a fully voice-controlled email assistant that lets you compose, edit, and send emails using natural speech, entirely hands-free. It intelligently finds the right contact, drafts emails using AI, reads them back for confirmation, and sends them seamlessly, all from your terminal. BEST PART, It completes the entire workflow within 2 minutes
